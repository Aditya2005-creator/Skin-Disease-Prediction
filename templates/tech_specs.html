Security Features

File Upload Security:

  • File type validation (JPG, PNG, JPEG only)
  • File size limit (16MB maximum)
  • Secure file handling
  • No executable file uploads

Data Privacy:

  • Local image processing only
  • No data transmission to external servers
  • Images stored temporarily in uploads folder
  • No user data collection

Deployment Information

Development Server:

python app.py # Runs on http://127.0.0.1:3000 # Debug mode enabled

Production Deployment:

gunicorn -w 4 -b 0.0.0.0:8000 app:app # Multi-worker production server # Configure reverse proxy (nginx)
